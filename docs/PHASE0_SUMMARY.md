## 这页记录什么

Phase 0 主要代表一段历史性探索：旧的 SwarmRouter 叙事、AgentGraph class 中心化叙事，以及 Shadow 相关设计线索。这里保留的内容只用于帮助读者理解演进背景，不用于定义当前实现边界。

## 这页不负责什么

- 不定义当前 Phase 1 runtime contract
- 不作为 CLI / HTTP 入口的最新说明
- 不作为 workflow schema 的规范来源
- 不作为 step / artifact / checkpoint 结果模型的权威解释
- 不作为当前官方示例或迁移基线

## 历史脉络概览

Phase 0 的文档曾经尝试把下列概念组织成一体化叙事：

- SwarmRouter 路由与竞标机制
- AgentGraph class 作为中心编排对象
- Shadow 作为扩展叙事与集成方向
- 记忆、检索、LLM provider、可视化等能力的聚合说明

这些内容可以帮助理解早期设计思路，但它们已经不能代表当前主线的正式边界。
