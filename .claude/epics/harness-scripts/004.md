# Task: 内置 validator 种子库

## Description

提供 4-6 个开箱即用的 builtin validator，覆盖文档/代码/前端三大主流 team 类型。
让用户不写 shell 就能挂上常用校验。

## Acceptance Criteria

- [ ] `shadowflow/runtime/validation_hooks/validators/` 目录建立 + registry 模式
- [ ] 至少实现：
  - [ ] `tsc-check` — 跑 `tsc --noEmit`，0 错才通过
  - [ ] `pytest-pass` — 跑 `pytest`，0 失败才通过
  - [ ] `lint-clean` — 跑 `ruff check .` / `eslint`，0 警告/错误
  - [ ] `chrome-console-clean` — 用 chrome-devtools MCP 抓 console，0 error 才通过（前端 team 必备）
  - [ ] `file-exists` — 简单存在性检查（如 "artifact 必须存在 dist/main.js"）
- [ ] 每个 validator 有自带 README + 配置 schema + 单元测试
- [ ] §005 UI 配置面板能从 registry 自动列出可选 validator

## Technical Details

- chrome-console-clean 实现需调用 chrome-devtools MCP 的 `list_console_messages` tool
- 所有 validator 接同一 `Validator.run(config, turn_ctx) -> ValidationResult` 接口
