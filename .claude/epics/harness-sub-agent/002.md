# Task: 角色契约编辑 UI（team-editor）

## Description

新建 team 编辑面板，让用户拿到自动生成的 team 后能 review + 修改任意 agent 的 RoleProfile
深度字段（responsibilities / constraints / handoff_rules / collaboration_contract）。
是替代下架的 `/builder` 的新形态——更聚焦"编 team 而非编 blueprint"。

- [ ] `src/pages/TeamEditorPage.tsx` 或 `src/components/team-editor/` 落地

- [ ] 展示该 team 全部 agent 的 RoleProfile，可点开任一 agent 编辑：
  - name / soul / role（基础字段，复用现有 CreateAgentModal 组件）

  - constraints（增删改 list）
