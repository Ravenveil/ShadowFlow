# Task: Runtime 注入器

## Description

turn 开始时把 team rule（enabled & severity=hard）拼成 markdown block 注入到 system prompt 前部，
让所有 agent 强制读到。

## Acceptance Criteria

- [ ] 找到 turn 启动代码（candidates: `shadowflow/runtime/turn_executor.py` /
      `server/src/routes/run-sessions.ts` 的 system prompt 拼装点）
- [ ] 加 `RulePackInjector.inject(team_id, system_prompt) -> system_prompt`
- [ ] 注入格式：`## 团队硬底线规则\n\n{rule.content}\n\n...` 在 system prompt 顶部
- [ ] severity='soft' 的 rule 注入但加标签 `[建议]`
- [ ] turn 元数据记录"本次注入了哪些 rule_id"，便于追溯
- [ ] 集成测试：mock team with 2 rules，验证 system prompt 含完整 rule 文本

## Technical Details

- 注入顺序：team rules → agent persona → user goal
- token 预算：rule 总长 > 2K 时截断 + 警告
