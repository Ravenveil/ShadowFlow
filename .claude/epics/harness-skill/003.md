# Task: 通用 SOP skill 种子

## Description

沉淀 3-5 个 ShadowFlow 内置 "通用 SOP skill" 作为种子，让新用户开箱即用。
对应文章 §1.9 主张"把高频固定动作沉淀成 Skill"。

## Acceptance Criteria

- [ ] `.shadowflow/skills/` 下新增 3 个种子 skill（每个含完整 SKILL.md + steps）：
  - **`sf-doc-sop`** — 文档 SOP：起草 → 自查 → 拼装 → 导出 PDF（用 deterministic step 调本地命令）
  - **`sf-code-sop`** — 代码 SOP：读代码 → 改代码 → 跑 lint → 报告 diff
  - **`sf-research-sop`** — 研究 SOP：搜文献 → 抽笔记 → 写综述 → 引用核对
- [ ] 每个 skill 至少 50% step 是 deterministic 类型（不全是 LLM step）
- [ ] 装包流程：用户从 SkillDropdown 一键启用其中之一用于其 team
- [ ] 文档化每个 skill 的目标用户 / 入参 / 产出 / 配合的 validation hook

## Technical Details

- 参考 BMAD-METHOD 现有 skill 结构 + 我们 001 扩展的 steps schema
- 与 [[harness-scripts]] 004 builtin validator 配合：每个 SOP 推荐挂哪几个 validator

## Dependencies

- 001 完成（steps schema 可用）
