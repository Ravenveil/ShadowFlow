# Task: Runtime injection + 权限策略门 + 调用审计

## Description

让 team 内的 agent 在 turn 执行期能调用 team-bound MCP server 工具，但所有调用走前置策略门
+ 全量审计。

## Acceptance Criteria

- [ ] turn 启动时 `MCPInjector.inject(team_id, agent_tools)` 把 team.mcp_servers 的工具列表合并到 agent 可用工具
- [ ] 每次 MCP 调用前查 permission：
  - `allow` 直通 + 记 audit
  - `ask` 暂停 turn + 弹用户确认（与现有 approval 机制复用）
  - `deny` 拒绝 + 记 audit
- [ ] 凭证从 secrets store 解析（不出现在 turn ctx / prompt 里）
- [ ] 所有 MCP 调用写 audit log：team_id / agent_id / server_id / tool / args_hash / result_status / timestamp
- [ ] 集成测试 mock 一个 MCP server 跑通三种 permission

## Technical Details

- 复用现有 ApprovalCard / Hermes 审批机制
- audit log 写到 `.shadowflow/audit/mcp-{date}.jsonl`
